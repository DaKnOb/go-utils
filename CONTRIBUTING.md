# Contributing

Contributions are more than welcome!

Requirements:
* Tests should be written
* No dependencies should be added unless necessary

#### Adding to, or modifying an existing package

1. Make your changes
1. Write any required tests (*highly encouraged*)
1. Open an MR

#### Creating a new package

1. Create the new package with a relevant name (e.g. for utilities based around strings, you would name it `stringutils`)
1. Write tests
1. Open an MR
